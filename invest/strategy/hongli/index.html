指数估值策略

中证红利 · 股息率档位策略

000922.CSI

1这个策略是什么

主信号是中证红利的股息率，并以 PE / PB 的十年分位、隐含 ROE 作交叉验证。把股息率划成六档，每档对应一个动作：股息率越高越便宜、买得越重，越低越贵、逐步止盈。「基数」同样指每月计划定投额，倍数为当月实际投入的比例。

股息率动作含义
≥ 5.8%重仓 ×3 基数极度便宜
5.0% ~ 5.8%加仓 ×2 基数明显便宜
4.5% ~ 5.0%定投 ×0.5 基数合理偏便宜
3.8% ~ 4.5%持有不动中性
3.5% ~ 3.8%预警，准备减仓偏贵
< 3.5%分批止盈 ×1 基数止盈区

2背后的逻辑，以及它是怎么来的

为什么用红利指数 + 股息率。中证红利本身就是以「高股息」为选样标准的策略指数，成分都是持续高分红的公司。而股息率的分子是企业真实派发的现金分红，比依赖会计利润、容易被调节的市盈率更稳、更难粉饰——对这类指数来说，它是一个更结实的估值锚。

估值方向为什么是反的。股价越低、分红不变，股息率就越高，意味着你用同样的钱能买到更多分红——所以股息率高 = 便宜 = 加仓，股息率低 = 贵 = 止盈。方向和 PE 策略其实一致，只是把锚从「市盈率」换成了「股息率」。

分位是怎么算的。本地用投资数据网 API 取十年股息率序列，分位定义为「历史上股息率低于当前值的比例」。由于股息率越高越便宜，所以分位越高，反而说明当前股息率在历史里偏低、当前越贵。再用 PE / PB 的十年分位、隐含 ROE 做交叉印证，避免单一指标误判。

档位阈值怎么定的。按红利指数的历史股息率分布划线——5.8% 以上是历史极高区（极便宜），3.8%~4.5% 是常态中枢，3.5% 以下进入历史偏低区（偏贵到止盈）。

3这个策略可能面临的风险

  • 股息率陷阱。股息率走高未必是「变便宜」，也可能是股价因基本面恶化而下跌；或者是成分股即将削减分红、而股价尚未反应——这时高股息是个假信号。
  • 分红的不确定性。整套逻辑依赖企业持续高派息。经济下行、盈利下滑或政策变化时，分红可能被下调，估值锚随之松动。
  • 成分集中。红利指数偏重银行、煤炭、交运等价值/周期板块，这些板块的系统性风险较大，并非充分分散。
  • 利率环境。当无风险利率大幅上行，高股息资产的相对吸引力下降，股息率可能长期停留在「偏贵」一侧而不回归。

4限制条件 / 什么情况下它不再成立

  • 历史可比性被破坏。十年分位假设历史与当下可比。若指数的成分或选样规则大改、整体股息特征变化，分位与阈值都会失效。
  • 分红政策系统性改变。若市场普遍转向少分红、多回购或留存，股息率这个锚本身就会失真。
  • 数据源问题。投资数据网 API 中断或口径调整，会导致信号停更或整体偏移。
  • 纪律前提。同样以「长期、纪律化、资金可持续」为前提；中途主观干预或停止投入，统计优势不再成立。
